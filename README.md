# TKB Home
This app adds support for TKB Home devices in Homey.

## Supported devices:
+ TZ65D - Dual Wall Dimmer
  - Right Switch only flow triggers
+ TZ67 - Wallplug Dimmer (all types)
+ TZ68 - Wallplug Switch (all types)

## Supported Languages:
* English
* Dutch (Nederlands)
